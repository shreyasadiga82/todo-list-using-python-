# 📝 Command-Line To-Do List (In-Memory)

A simple command-line to-do list application built in Python. Users can add, 
remove, and view tasks while the program is running. This is **Project #2** 
in my daily coding practice series.

## 📋 Description

This program runs a menu-driven loop where the user can manage a list of 
tasks entirely in memory (nothing is saved to a file — once the program 
closes, the list resets). It's a step up from Project #1, focusing on 
working with lists and organizing code into functions.

## ✨ Features

- Add a new task
- View all current tasks (numbered list)
- Remove a task by its number
- Mark a task as done (optional stretch feature)
- Simple menu loop that keeps running until the user exits

## 🧠 Concepts Practiced

- Lists (`append`, `remove`, indexing)
- Functions (breaking logic into reusable pieces)
- Menus and program flow using `while` loops
- User input handling and basic validation

## 🛠️ Tech Stack

- Python 3

## 🚀 How to Run

1. Clone this repository
```bash
   git clone https://github.com/your-username/todo-list-cli.git
```
2. Navigate into the project folder
```bash
   cd todo-list-cli
```
3. Run the script
```bash
   python todo.py
```
